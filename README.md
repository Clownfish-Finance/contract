# Clownfish Finance Contract
An innovative DeFi platform designed to provide comprehensive credit account management and leveraged trading capabilities.
### A Guide to Getting Started
1. Install Dependencies  
`npm install`
2. Compile Contracts  
`npx hardhat compile`
3. Run Tests
`npx hardhat test`
